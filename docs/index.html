Simple API Docs¶

simplest__bsod(self)¶

Call`s simple bsod using simle cmd command that is called by os module in python For fact ths command is working on broswer : \.globalrootdevicecondrvkernelconnect You can call it using :

bsod.simplest__bsod()

taskmanger_way(self)¶

Call`s simple bsod using simple cmd command that is called by os module in python For fact ths command is on powershell : runas /user:administrator TASKKILL /IM svchost.exe /F Also runas is just for as admin rights. You can call it using :

bsod.taskmanger_way()

use_external(self)¶

Runs simple tool by Microsoft and let you manualy run bsod. You can run it as :

bsod.use_external()
